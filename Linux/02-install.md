# Linux 环境安装

安装Linux环境有几种方式：

- 使用虚拟机创建虚拟环境安装Linux系统
- 使用Docker来创建Linux环境容器

因本地已经安装了Docker，且Docker安装Linux环境非常方便，所以我将采用Docker来安装Linux环境。

```bash
$ docker run -it centos 
```
